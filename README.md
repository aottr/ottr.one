# ottr.one

## Contribute
1. Install node.js
    ```
    brew install node
    ```
1. Install dependencies
    ```
    node -i
    ```
1. Start dev server
    ```
    npm run dev
    ```
1. Build app
    ```
    npm run build
    ```

## IE11 compatibility

If you..for some weird reason... want to make the app IE11 compatible you'll have to set the `target` in `tsconfig.json` back to `es5`.
